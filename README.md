# Track Tasks

Track Tasks is a simple client-side application built with React and Tailwind CSS that helps you manage your tasks using Web Storage API's Local Storage.

View your app in AI Studio: https://ai.studio/apps/7a1c0602-eaa5-4d2d-9faa-02a330f84d7f

## Run Locally

**Prerequisites:** Node.js (version 20 or higher recommended)

1. Clone the repository (if you haven't already):
   `git clone https://github.com/sheeeng/track-tasks.git`
   `cd track-tasks`
2. Install dependencies:
   `npm install`
3. Run the development server:
   `npm run dev`
4. Open your browser and navigate to the URL provided in the terminal (usually `http://localhost:3000` or `http://localhost:5173`).
